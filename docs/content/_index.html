{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Greple is a text search CLI tool with some distinguishable features, including multiple keywords by lexical expression, line-across match, match area control, flexible blocks, multi-byte character friendliness, multiple colors, etc. It is extensible by modules and you can make your own module for specific purpose. This makes Greple as a framework for various text processing. {{% /blocks/lead %}} {{% blocks/lead color="secondary" %}} Greple はテキスト検索ためのコマンドラインツールで、次のような特徴を持っています。 複数キーワード検索、 複数行に渡るマッチ、 柔軟なデータブロック、 マルチバイト文字処理、 マルチカラーによる多彩な出力、 等。 また、モジュールによる高い拡張性を持ち、目的に応じた独自モジュールを作ることができます。 このことにより、Greple は様々なテキスト処理のためのフレームワークとして利用可能です。 {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/kaz-utashiro/Greple-Home" %}} We do a [Pull Request](https://github.com/kaz-utashiro/greple/pulls) contributions workflow on **GitHub**. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-twitter" title="Follow us on Twitter!" url="https://twitter.com/uta46" %}} For announcement of latest features etc. {{% /blocks/feature %}} {{< /blocks/section >}}
