{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Greple is a text search command similar to Unix grep with some distinguishable features, including multiple keywords by lexical expression, line-across match, match area control, flexible blocks, multi-byte character friendliness, multiple colors, etc. It is extensible by modules and you can make your own module for specific purpose.
